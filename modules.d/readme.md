# modules.d 询问交互界面模块根级目录

# 该模块下的脚本，仅存放需要选项的根级交互界面逻辑，功能全调用 shell_scripts 对应脚本

```sh
# 假设 文件名为 system.sh
### === 主菜单 === ###
system_menu(){}
```

# modules.d/xxx.d 询问交互界面模块子级目录

# 该模块下的脚本，仅存放需要选项的子级交互界面逻辑，功能全调用 shell_scripts 对应脚本

```sh
# 假设 文件名为 system_reinstall.sh
### === 主菜单 === ###
system_reinstall_menu(){}
```
